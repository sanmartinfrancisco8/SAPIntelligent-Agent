# **App Name**: SAP B1 Companion

## Core Features:

- Replicated Navigation: Mimic the SAP Business One desktop navigation with a main menu replicating the SBO module hierarchy.
- Module and Functionality Display: Display module overviews, key features, and version-specific updates in a central panel, mirroring SBO's functionality.
- AI Chat Assistant: Implement an AI assistant that answers real-time queries using the app's knowledge base and the ability to retrieve extra knowledge using tool use.
- Interactive Mind Maps: Dynamically generate mind maps representing the module's conceptual structure.
- Process Flow Diagrams: Generate visual diagrams illustrating process flow and document traceability for selected functionalities.
- Concise Summaries: Synthesize key information for the module and functionality currently being consulted.
- Integrated Login Simulation: Simulate the SAP Business One login process requiring User ID and Password.
- Spanish Language Support: The entire application in Spanish

## Style Guidelines:

- Primary color: A deep blue (#1A237E) to evoke trust, authority, and stability.
- Background color: A light gray (#F5F5F5), offering a neutral backdrop that mimics professional software environments and ensures readability.
- Accent color: A vibrant teal (#00BCD4) for interactive elements and highlights, providing contrast and signaling interactivity without overwhelming the user.
- Headline font: 'Poppins', a geometric sans-serif, to give a precise, contemporary feel to the titles.
- Body font: 'Inter', a grotesque-style sans-serif to ensure clear readability, even in long sections of text.
- Use flat, professional icons that reflect the ERP system's modules and functionalities. Consistent style for easy recognition.
- Mimic SAP Business One's clean and structured layout, with a left-side menu and a main content panel for displaying information.
